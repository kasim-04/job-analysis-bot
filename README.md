# Job Analysis Bot

Telegram-бот для анализа зарплат по различным профессиям.

## 📌 Описание

Этот проект представляет собой Telegram-бота, который помогает узнать диапазон зарплат для интересующей профессии в Москве.

<video src="media/preview.mp4" controls width="100%"></video>

**Важно**: 
- Проект создан исключительно в учебных целях
- Не предназначен для коммерческого использования
- Использует публичное API hh.ru ([официальная документация](https://github.com/hhru/api))

## 🔍 Как это работает

1. Пользователь отправляет боту название профессии (например, "Python-разработчик").
2. Бот собирает вакансии с указанной зарплатой.
3. Рассчитывается диапазон зарплат:
   - нижняя граница — 25-й процентиль,
   - верхняя граница — 75-й процентиль.
4. Бот отправляет результат пользователю.

## ⚙️ Установка и запуск

1. Клонируйте репозиторий.
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Создайте файл .env в корне проекта со следующим содержимым:
```
TOKEN='ваш_токен'
USER_AGENT='ваш_user_agent'
```
4. Запустите бота:
```
python run.py
```
